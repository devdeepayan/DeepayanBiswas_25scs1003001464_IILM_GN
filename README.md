# AI Question Generator

## 📌 Description
This project generates questions automatically from a given paragraph using the T5 (Text-to-Text Transfer Transformer) pre-trained model.

## 🧠 Model Used
T5 (valhalla/t5-small-qg-prepend)

## 🛠️ Technologies Used
- Python
- Transformers (Hugging Face)
- PyTorch

## ⚙️ How to Run
1. Install dependencies:
   pip install transformers torch sentencepiece

2. Run:
   python ai_question_generator.py

## 🎯 Output
Generates multiple questions from input text.
